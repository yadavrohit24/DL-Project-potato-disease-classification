# 🥔 Potato Disease Classification using Deep Learning

An end-to-end Deep Learning project that classifies potato leaf diseases using image processing and Convolutional Neural Networks (CNN). This system helps in early disease detection, enabling better agricultural decision-making and improved crop yield.

---

## 🚀 Project Overview

Potato crops are vulnerable to diseases such as Early Blight and Late Blight, which can significantly reduce productivity. This project leverages deep learning to automatically classify potato leaf images into disease categories, assisting farmers and researchers in accurate diagnosis.

---

## 🎯 Objectives

- Detect potato leaf diseases using image classification
- Automate disease identification process
- Improve agricultural productivity through AI solutions

---

## 🧠 Model Architecture

- Model Type: Convolutional Neural Network (CNN)
- Framework: TensorFlow / Keras
- Input: Potato leaf images
- Output Classes:
  - Early Blight
  - Late Blight
  - Healthy

---

## 📂 Dataset

The model is trained on a labeled dataset of potato leaf images.

- Classes:
  - Healthy
  - Early Blight
  - Late Blight
- Dataset Source: PlantVillage (Kaggle)
